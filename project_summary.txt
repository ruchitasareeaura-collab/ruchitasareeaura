================================================================================
📝 CUSTOMIZATION:
================================================================================

Update Prices:
- Edit js/products.js
- Change price & originalPrice fields

Change Contact Info:
- Update all HTML files (footer section)
- Search for phone, email, address

Replace Images:
- Drop new JPGs in /images/ folder
- Keep naming: saree-01.jpg, saree-02.jpg, etc.
- Size: 800×1200 for products, 1920×1080 for hero

Update Shipping:
- Edit js/checkout.js
- Change deliveryCharge values

Add Products:
- Add to products array in js/products.js
- Include all required fields
